# Básico de Markdown

Markdown é uma linguagem de marcação leve e fácil de usar. Aqui estão alguns elementos básicos:

## Títulos
Use `#` para criar títulos:
```markdown
# Título 1
## Título 2
### Título 3
```

## Texto
- **Negrito:** `**texto**` ou `__texto__`
- *Itálico:* `*texto*` ou `_texto_`
- ~~Riscado:~~ `~~texto~~`

## Listas
- Lista não ordenada:
    ```markdown
    - Item 1
    - Item 2
    - Item 3
    ```
- Lista ordenada:
    ```markdown
    1. Primeiro
    2. Segundo
    3. Terceiro
    ```

## Links e Imagens
- Link: `[texto](url)`
- Imagem: `![alt](url)`

## Código
- Inline: `` `código` ``
- Bloco:
    ```markdown
    ```
    Código em bloco
    ```
    ```

## Tabelas
```markdown
| Cabeçalho 1 | Cabeçalho 2 |
|-------------|-------------|
| Linha 1     | Valor 1     |
| Linha 2     | Valor 2     |
```

## Citações
```markdown
> Isso é uma citação.
```

## Separadores
```markdown
---
```
